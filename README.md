# go-wasm-captcha-demo
Example of Go WASM - JS Interaction

# Prerequisites
* Go
* Node.JS 10+ (e.g. https://github.com/nvm-sh/nvm)

# Compiling
```bash
git clone https://github.com/DirectX/go-wasm-captcha-demo.git
cd go-wasm-captcha-demo
./compile.sh
```

# Running Web Server
```bash
npx lite-server
```
